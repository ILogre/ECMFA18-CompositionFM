You will find the following files :
- compositions.cvs : characterization matrix of the operators wrt. our criteria
- CSV2FM_Composition.java : dumb code to transform this csv to feature models
- compositions_fms_script.fml : the list of FMs (one per operator) plus a command to merge them
- fmmerged.fml : the textual, readable, output of the merging operation
- compositions.treeml : the Familiar graphical representation containing the 81 constraints
